# HABEIS CENTRO EDUCACIONAL - Sistema de Gestão de Certificados

## Sobre o Projeto

Este projeto consiste em um sistema web para gerenciamento e emissão de certificados de cursos preparatórios para concursos públicos. A plataforma permite que concurseiros e outros profissionais obtenham certificados válidos para uso em inscrições de concursos e para aprimoramento profissional.

O sistema é focado em quatro áreas principais:
- Gerenciamento de Cursos
- Gerenciamento de Alunos
- Sistema de Matrículas
- Emissão de Certificados

## Tecnologias Utilizadas

- Frontend: HTML, CSS, JavaScript
- Backend: (a definir)
- Banco de Dados: MySQL
